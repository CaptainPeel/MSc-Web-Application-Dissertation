About This Page

This web application is part of an ongoing series of development aimed at created a new online based educational tool whereby users can increase or harness new skills around the basis of website development. Within this web application users have FREE access to a live code editor as well coding tutorials where they can learn about differing languages and how to development with these languages

  • CSS
  • JavaScript

Frequently Asked Questions

  • What is the purpose of this website application?

    The main premise of this proposed project is to create a piece of software/website that users can access in order to view online tutorials or test live code around the basis of website programming, for instance within the website there would be live code that could be run where the user can see a breakdown of what the code is outputting as well as the option to live edit the test code on the website so that the user can grasp a better understanding of the code. In addition to this feature, I also plan to implement the use of tutorials that will assist the user with programming or developing their website. The main focus of this proposed project would be to culminate a plethora of features to translate into the learning of HTML, CSS, JavaScript and other various programming languages..

  • Why is this application free?

    This application is free because I want to be able to provide an online digital learning platform to a plethora of users with content that is not locked behind a paywall.

  • Do you plan to add support for more programming languages?

    The future scope of this project and subsequent application would be to provide more content to various programming languages.

  • Do you plan to add monetary features?

    There is no current plan to add monetary features to this application. This application shall remain free for as long as possible.

  • Can we donate?

    Donations are not currently being taken however, thank you for your support.
