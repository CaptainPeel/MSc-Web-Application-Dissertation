HTML Tutorials

Learning Headings

HTML headings are titles or subtitles that you want to display on a webpage.

Learning Paragraphs

A paragraph always starts on a new line, and is usually a block of text. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Learning Images

Images can improve the design and the appearance of a web page. Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image.

Learning Tables

HTML tables allow web developers to arrange data into rows and columns.

Learning Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

Learning HTML in JavaScript

JavaScript makes HTML pages more dynamic and interactive.

Learning Hyperlinks

TML links are hyperlinks. You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand.

Learning Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

Learning Videos

The HTML video element is used to show a video on a web page.

Learning Canvas

A canvas is a rectangular area on an HTML page. By default, a canvas has no border and no content.
